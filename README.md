# Video Monitor Hub

Video Monitor Hub is a web application for registering security cameras, viewing recent motion recordings, and managing saved video updates. The project combines a Laravel web application, a PostgreSQL database, and a Python/OpenCV processor for motion detection and video recording.

## Main Features

- User registration, login, and logout.
- Camera registration with name, access URL, and local file storage path.
- Camera listing with view, edit, and delete actions.
- Recent camera updates page with recorded video logs.
- Modal video playback for recordings generated by motion detection.
- Python processor that reads registered cameras, detects motion from camera streams, records video files, and stores update logs.

## Running With Docker

Start the application stack:

```bash
docker compose up -d --build
```

On first startup, the Laravel container prepares the application runtime by:

- copying `laravel-app/.env.example` to `laravel-app/.env` when `.env` does not exist;
- applying Docker database defaults to that new `.env`;
- installing Composer dependencies into a Docker volume;
- generating `APP_KEY` when needed;
- installing Node dependencies into a Docker volume;
- building the Vite frontend assets when needed.

The Laravel application is exposed at:

```text
http://localhost:8000
```

PostgreSQL is exposed on port `5432`. The default Docker database values are:

```text
DB_DATABASE=video_monitor
DB_USERNAME=postgres
DB_PASSWORD=secret
```

Inside `laravel-app/.env`, configure the Laravel database connection for Docker:

```text
DB_CONNECTION=pgsql
DB_HOST=database
DB_PORT=5432
DB_DATABASE=video_monitor
DB_USERNAME=postgres
DB_PASSWORD=secret
```

Then run the Laravel setup commands inside the application container:

```bash
docker compose exec laravel-app php artisan migrate
```

To reset the database during development, remove the database volume and run migrations again:

```bash
docker compose down -v
docker compose up -d --build
docker compose exec laravel-app php artisan migrate
```
